1. **클라우드 네이티브 개발**: 도커, 컨테이너, 쿠버네티스
2. **풀스택 개발**: HTML5, React + Spring Boot
3. **DevOps 실무**: CI/CD, 자동화, 모니터링

1. **단계별 순차 학습**: 1단계부터 6단계까지 순서대로 진행
2. **병행 학습**: 7단계(도커, AWS)와 8단계(프로젝트)는 3-6단계와 병행
