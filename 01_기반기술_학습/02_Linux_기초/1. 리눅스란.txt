1. 리눅스(Linux)
    1) 윈도우나 맥OS를 운영체제(OS)라고 애기함 
        - 컴퓨터라는 하드웨어를 작동시키기 위한 소프트웨어 
    2) 리눅스도 비슷하게 여러 운영체제 중 하나임 

2. 다양한 리눅스의 배포판 (Distribution)
    1) 사람들이 가장 많이 쓰는 대중적인 우분투(Ubuntu)

3. 리눅스 환경 구축 (AWS EC2)
    1) Free Tier    

4. 터미널 (Terminal)
    1) 컴퓨터를 직접 제어하기 위한 명령어를 입력할수 있는
       창   
    2) git bash, cmd, ps, modaxterm...

    3) ubuntu@ip-172-31-38-65:~$    
        - @ 앞에 있는 값인 ubuntu는 사용자 이름  
        - @ 뒤에 있는 값인 ip-172-31-38-65는 호스트 이름,
          즉 컴퓨터 이름임.
            - 기본 값으로 해당 컴퓨터의 IP 주소로 이름이 
              지어져있음.        
        - ~ 
            - 현재 들어와있는 경로를 뜻함
            - 사용자의 홈 디렉토리를 의미함 
        - $
            - 사용자의 명령 입력을 기다리는 표시임.  

    4) 기본 조작법 
        - 리눅스에서 작업 중단, 작업 취소 
            - Ctrl + c 
        - 터미널 화면 깔금하게 지우고 싶을 때 사용 
            - clear    
        - 코드 복사하기 / 붙여넣기 
            - Ctrl + Insert : 복사하기
            - Shift + Insert : 붙여넣기   

5. 리눅스에서의 파일(file), 디렉터리(directory)    
    1) 윈도우, 맥에서 파일, 폴더(folder) 개념
    2) 리눅스에서는 폴더라고 부르지 않고, 디렉토리라고 부름.
    3) 리눅스에서는 모든 걸 파일로 간주함 
        - 일반 파일 : 텍스트, 동영상 파일 등
        - 디렉터리 파일 : 윈도우에서의 폴더와 같은 개념

6. 리눅스의 기본 디렉터리 구조 
    1) 현재 디렉토리 경로 조회 (pwd)
        - 경로(path)는 특정 파일이 어디에 있는 지를 
          설명하는 주소 
    2) 디렉터리 이동 (cd)    

7. 리눅스 기본 폴더 구조 
    1) 리눅스의 최상위에 있는 디렉토리(/)를 
       루트 디렉토리라고 함. 
       - 최상위 디렉토리라고 함.
    2) 리눅스에서 경로를 표현할 때 슬래시(/)를 활용함
       - /home/ubuntu/aws-ec2-express/
       - 참고로 윈도우는 역슬래시(\) 사용
         - C:\Program Files\Java     

8. 현재 디렉토리 내부에 있는 파일 조회 (ls)
    1) 디렉토리 구조 확인
    2) list의 줄임말

9. 파일 종류 확인 (ls -l)
    1) 맨 앞 한 글자가 파일 종류를 나타냄
        - - : 일반 파일
          d : 디렉토리 
          l : 링크 (바로가기)

10. 숨김 파일 조회 (ls -a)
    1) 리눅스에서는 점(.)으로 시작하는 파일명을 가진 파일은 
       저절로 숨김 파일로 인식해 처리됨.
    2) 숨김 파일을 조회하려면 ls -a 명령어를 입력해 줘야함.

11. 리눅스 명령어들의 패턴 - 옵션(Option)
    1) 예
        - ls # 현재 디렉토리 내부 파일 조회 
        - ls -l  # 현재 디렉토리 내부 파일 조회 + 자세히 정보 조회 
        - ls -a  # 현재 디렉토리 내부 파일 조회 + 숨김 파일 조회       
    2) 하이픈(-)으로 시작하는 옵션을 추가함 
    3) 옵션의 특징 
        - short option
            - 하이픈 한 개(-)로 시작함  
        - long option
            - 하이픈 두 개(--)로 시작함 
        - https://man7.org/linux/man-pages/man1/man.1.html
