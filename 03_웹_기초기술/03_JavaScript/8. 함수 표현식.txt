1. 익명 함수 
    1) 이름이 없는 함수를 말함.
    2) 익명 함수는 함수 자체가 식(expression)임.
        - 함수를 변수에 할당할수 있음 
        - 다른 함수의 매개변수로 사용할 수 있음.

2. 즉시 실행 함수 (Immediately Invoked Function Expression, IIFE)
    1) 한 번만 실행하는 함수라면 함수를 정의하면서 
       동시에 실행할 수 있음.
    2) 주로 변수의 스코프를 제한해서 코드 사이에 충돌이 
       생기는 것을 방지하려고 사용함 
    3) 한 번 실행하고 끝내므로 전역 변수 충돌이 생기지 않음.
    4) 기본형 
        - (function() {
            명령 
        }());                  

        - (function(매개변수) {
            명령
        }(인수));

3. 화살표 함수 
    1) ES6 버전부터 => 표기법 사용함 
    2) 함수 선언을 좀 더 간단하게 작성 가능 
    3) 기본형 
        - (매개변수) => { 함수 내용 }  
    4) 매개변수가 없을 경우 
        - 예)
          const hi = function() {
            return "안녕하세요";
          }            

          const hi = () => { return "안녕하세요"; };

          const h1 = () => "안녕하세요";

    5) 매개변수가 1개인 경우 
        - 매개변수의 괄호는 생략 가능함 

    6) 매개변수가 2개 이상인 경우
