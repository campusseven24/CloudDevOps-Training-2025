1. 변수의 종류
    1) 지역 변수 
        - 메서드에 선언, 매개변수도 지역 변수의 한 종류임 
    2) 멤버 변수 (필드)
        - 클래스에 선언   

2. 변수의 값 초기화 
    1) 지역 변수 : 수동 초기화 
        - 지역 변수는 항상 직접 초기화 해야 함.
    2) 멤버 변수 : 자동 초기화    
        - 숫자(int) = 0, boolean = false
        - 참조형 = null 
            - null 값은 참조할 대상이 없다는 뜻으로 사용됨.
        - 개발자가 초기값을 직접 지정할 수 있음.  

3. null 
    1) 참조형 변수에는 항상 객체가 있는 위치를 가리키는 참조값이 들어감.
    2) 아직 가리키는 대상이 없거나, 가리키는 대상을 나중에 입력하고 싶은경우
    3) 참조형 변수에서 아직 가리키는 대상이 없다면 null이라는 
       특별한 값을 넣을 수 있음.
       - null은 값이 존재하지 않는, 없다는 뜻임.

4. NullPointerException   
    1) 주소가 없는 곳을 찾아갈 때 발생하는 예외임.   

    2) 멤버 변수와 null
