1. 지역 변수와 스코프 
    1) 변수는 선언의 위치에 따라 지역 변수(Local Variable), 
       멤버 변수(클래스 변수, 인스턴스 변수) 분류됨 

    2) 지역 변수 
        - 특정 지역에서만 사용할 수 있는 변수라는 의미임.
            - 변수가 선언된 코드 블록 ({})임. 
        - 지역 변수는 자신이 선언된 블록안에서만 생존하고,
          자신이 선언된 코드 블록을 벗어나면 제거됨 
            - 이후에는 접근 불가함   

    3) 스코프(scope)
        - 변수의 접근 가능한 범위                     

    4) 스코프 존재 이유 
        - 비효율적인 메모리 사용 방지
            - 효율적인 메모리 사용 가능 
        - 코드 복잡성 증가 방지
