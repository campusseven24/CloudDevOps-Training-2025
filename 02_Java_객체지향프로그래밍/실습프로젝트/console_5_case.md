```
상품명을 입력하세요: 블루투스 이어폰 [사용자 입력]

재고 수량을 입력하세요: 25 [사용자 입력]

상품이 추가되었습니다: 블루투스 이어폰
```

## 예시 2: 장바구니에 상품 추가 후 주문하기

```

1. 상품 관리

=== 주문 관리 ===

2. 고객 주문 내역 보기
0. 메인 메뉴로 돌아가기

고객 ID를 입력하세요: C001 [사용자 입력]

=== 김철수님의 장바구니 ===
- 무선 이어폰 x 2개 = 300,000원

장바구니 합계: 300,000원
```
